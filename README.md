# NC Analytics and AI Day 2026

Conference website for **NC Analytics and AI Day 2026**, hosted by the North
Carolina INFORMS Chapter and the NC State Edward P. Fitts Department of
Industrial and Systems Engineering.

- **When:** Friday, September 25, 2026
- **Where:** Duke Energy Hall, James B. Hunt Jr. Library, NC State University
  (1070 Partners Way, Raleigh, NC 27606)

## Files

- `index.html` — main conference homepage
- `compare.html` — internal page for comparing hero image options
- `styles.css` — shared stylesheet
- `hero.jpg` — current hero image (Hunt Library exterior, angled view)
- `hero-hunt1.jpg`, `hero-hunt2.jpg`, `hero-night.jpg` — alternate hero options
- `conference_info.docx` — source content document

## Running locally

From the project directory:

```
python -m http.server 8000
```

Then open <http://localhost:8000/index.html>.

## Hosting

The site is a static HTML/CSS site with no build step, suitable for GitHub
Pages, Netlify, Cloudflare Pages, or any static host.
